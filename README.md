# AUREL Motion

AUREL Motion is a luxury watch-inspired front-end concept built as a polished, fully responsive landing page. The project combines bold visual storytelling with interactive UI details such as theme switching, smooth reveal animations, a search overlay, a mobile menu, a rotating showcase, and a 360° watch viewer.

## Live Demo

**Website:** https://watch-ui-smoky.vercel.app

## Preview

The interface is designed around a high-end horology aesthetic with:

- cinematic hero section
- featured watch collections
- brand story section with animated stats
- ambassador / partner showcase
- interactive product carousel
- latest news cards
- newsletter signup form
- dark / light theme toggle
- responsive navigation and search modal

## Built With

- **HTML**
- **CSS**
- **JavaScript**

## Key Features

- **Responsive design** that adapts across desktop, tablet, and mobile devices
- **Theme toggle** with persistent styling support
- **Animated UI** using reveal effects, parallax motion, and counter animations
- **Interactive showcase** with next/previous controls and auto-play
- **360° viewer** for a more immersive product experience
- **Mobile navigation** and search panel for smaller screens
- **Dynamic content rendering** for collections, partners, and news cards
- **Luxury-inspired visual design** with layered gradients, glassmorphism, and premium typography

## Getting Started

If you want to run the project locally:

1. Clone the repository
   ```bash
   git clone https://github.com/Dnyaneshadkar9696/Frondend-Assignment.git
   ```

2. Open the project folder
   ```bash
   cd Frondend-Assignment/FrontEnd\ Assignment
   ```

3. Open `index.html` in your browser

   You can simply double-click the file or use a local development server.

## Recommended Local Server

For the best experience, use a local server instead of opening the file directly:

```bash
npx serve .
```

Or with VS Code, use the **Live Server** extension.

## Customization Ideas

You can easily adapt the project by updating:

- product data in `script.js`
- brand content in `index.html`
- color palette, layout, and animations in `styles.css`
- images and assets inside the `assets/` folder
